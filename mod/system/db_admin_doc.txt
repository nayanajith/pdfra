Create/Recreate tables and upgrade from previous version
-----------------------------------------------------------------------
Please select the action you want to do for each table and press Create/Re-Create to execute the request

### Create/Recreate

* CREATE
* RECREATE

> Rename(backup) the existing table with suffixing the current date if it contains any data and create a new empty table.

### Upgrade/Migrate

If the version of the database is less than the system schema you should upgrade the database to the new version
